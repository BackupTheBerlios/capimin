Editing the fax send queue

You can modify your outgoing fax by directly changing the values in the form.
To commit this changes, select the fax by clicking in the checkboxes (left on the list) and then press button "change".

List columns:
Checkboxes ID Faxnumber Addressee  Tries Next try Subject
Commands:
Select All Invert selection Change Change & Send 1hlater Delete

Checkboxes
Here you can select the fax jobs you want to modify/delete (the action is chosen by the command buttons described below)

ID
Every Job (fax/voice) has its own unique id. The id is created and increased automatically when you create new fax/a fax/voice call is received

Fax number
Destination fax number. Capimin allows you to use the following helper chars: +-()/ and space (empty) char.

Addressee
Name or description for the address. (optional)

Tries
Shows, how many times CapiSuite has tried to send the fax (the maximum number of tries before the fax is moved to the failed dir, is set in the CapiSuite config

Next try
Here you can read/set the send time of the fax. The date values (year, month, day) must be set, if hour and/or minute is missing, the will be set to "0"

Subject
Here you can read/set a subject/description for the fax job.

Select all and Invert selection
JavaScript helper functions to select all faxes or invert your current selection
(JavaScript must be enabled in your browser to use it)

Change
changes the selected faxes.
Note: If you don't have any faxes selected (with the checkboxes) but you have edited some data, nothing will change.

Change & Send
Applies the changes made to the selected faxes and sends them immediately (changes the start time
to the current time, so that capisuite will send the faxes, when it checks the queue)

Delete
removes the selected files from the send queue

1h later
Add one hour to the start time for the selected faxes.
